OpenGL Rendering Engine

Designed and built a 3D rendering engine on top of OpenGL, with a full scene graph, shader pipeline, and resource management system built for extensibility. Features hierarchical transforms with dirty-flag propagation, a hot-reloadable shader compilation pipeline, reference-counted resource lifetimes for textures and meshes, draw call batching to minimize GPU state changes, and a camera system supporting both perspective and orthographic projections with Phong lighting.

Custom Compiler & Language

Wrote a complete compiler toolchain for an original programming language — from raw source text to executable output, with no parser frameworks involved. Includes a hand-written lexer, a recursive-descent parser that produces a typed AST, a scope resolver and type checker, and a code generator targeting either a bytecode VM or native output. The language features static typing, first-class functions, and a clean expression grammar.

Lexer Parser AST Type Checking Code Generation Bytecode VM
